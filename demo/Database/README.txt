Instrucciones
=============
Para crear la Base de Datos hay que ejecutar el comando DropAndCreateZoosManagementSystemDatabase.cmd.


Requerimientos
==============
* Microsoft® SQL Server® 2008 Express with Advanced Services (http://www.microsoft.com/downloads/details.aspx?familyid=B5D1B8C3-FDA5-4508-B0D0-1311D670E336)
